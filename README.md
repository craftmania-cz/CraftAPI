### Jak spustit na localhostu
```shell
git clone https://github.com/craftmania-cz/craftapi.git
cd craftapi
npm install
```

Poté stačí vyplnit v `config/default.json.example` přepsat na `default.json` a vyplnit.

```shell
npm run develop
```
